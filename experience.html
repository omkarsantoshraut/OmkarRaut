  • Organization: The Sparks Foundation

  • Role: Data Science & Business Analytics

  • End Date: March 2021
I have an interest in the field of data science, and hence I had started my data science journey from the one month internship in the field of data science. Data science is a mixture of mathematics and logic, machine learning, data analytics, data visualization, etc. I had learnt about them but I wanted to get some experience by working on the projects and I got a chance to do an internship at The Sparks Foundations. Along with my learning process, I have done the tasks provided by the team from the organization. I would like to list down the tasks in brief.

  • Prediction using supervised machine learning:

    The problem statement was very simple. I had to predict the percentage of student in an exam based on the number of hours he/she has done the study. They had provided the test data and it had only two attributes such as nunber of hours of study and the percentage in an exam. I had to make a machine learning model that can predict the percentage of a student. Also, I had to submit the answers of some questions. They had given some question and based on that information I had to use the trained model and give the answers of that questions. That's all. This was the first very simple task that I had done. I had made a model by using simple linear regression from scratch as well as using sklearn library.

  • Prediction using unsupervised machine learning:

    The second problem was very straightforward. I had given the iris dataset. I had to use only input features to train the model and find out the optimized clusters in the given data i.e. I had to find out the number of classes of flower in the data. Also, I had to visualize it by making the graph. I had used KMeans clustering algorithm to find the number of classes of flower. I have started from 1 to 10 as a number of classes and based on that I had developed the KMeans models and store the error (mean squared error) of each iteration. Then I had plotted the graph number of classes VS error. I had got 3 as an optimized value of classes. Then I had developed a KMeans model by using 3 value of clusters. After that I had used matplotlib library to visualize it and shown the clusters with different colors.

  • Exploratory Data Analysis - Retail

    In this task, I had to make a dashboard based on the given data as a part of data analytics. I had used Tableau and made an interactive dashboard to visualize the given data. Now, I have learnt a new technology to make an interactive dashboards in python only. The name of the technology is Dash, which is developed by Plotly. It is very easy to use Dash and the main thing is that we can use it in any web framework. I have recently used it with Flask framework for on the projects.

  • Exploratory Data Analysis - Terrorism

    This task also very similar to the above. I had given a dataset in csv format. I had to do the data cleaning first, and then I had to do the exploratory data analysis. I had imported the csv file in the jupyter notebook, and then I have done data cleaning. After that I had saved the new data into the another csv file. Then I had imported that file into Tableau and had done the data analytics. There were some question I had to submit after building an interactive dashboard. I had done with that also.

  • Exploratory Data Analysis - Sports

    Again the same think I had to do. The only difference was that the dataset was different than the previous one. I had done with it using the Tableau and the python script.

  • Prediction using Decision Tree Algorithm

    In this problem, I had to use decision tree classifier and train a machine learning model that will classify the thing after providing a new input. I had done with it, and at last I had to make a diagram of decision tree. I had learnt it first, and then I was able to make it using python only.

  • Timeline Analysis : Covid-19

    In this problem, they had provided the COVID-19 dataset. I had to do a timeline analysis of the data. I had make an interactive dashboard in Tableau using that data with respect to the time. It was very interesting to work on the current problems. I was like a real world problem, I had enjoyed the duration of working on this problem.
